## 競技プログラミングのライブラリ

各アルゴリズムはクラスで実装されており, 実際に用いる際にはそのクラスと依存ライブラリ等をコピーして使用する. どこからコピーすれば良いかはコメントアウト部分を見ればよく, 依存ライブラリは`// Need start`から`// Need end`までの部分を, クラスは`// Copy start`から`// Copy end`までの部分を切り取れば良い.

* 作成済み
    * アルゴリズム
        * Dijkstra
        * Bellmanford
        * Warshall-Floyd
        * 二部グラフの最大マッチング
        * スライド最小値
    * データ構造
        * Union-Find
        * 抽象化BIT
        * 抽象化セグメント木
    * 幾何
        * 二次元座標の構造体
        * 凸包（グラハムスキャン）
    * 数学
        * 行列
        * 組み合わせ
    * 文字列
        * KMP
